How to connect Google Sheets (one-time setup, ~5 minutes)

1. Go to sheets.google.com → create a new blank spreadsheet.
2. Click Extensions → Apps Script.
3. Delete all existing code, paste the code from the box below, then click 💾 Save.
4. Click Deploy → New deployment → Web app.
5. Set Execute as: Me · Who has access: Anyone.
6. Click Deploy → copy the Web App URL → paste it below.
